# ✨ Lightroom Seoul <br>인터렉티브 사이트 클론 코딩

GSAP, ScrollTrigger를 사용하여 인터렉션 구현에 중점을 두었고,
SCSS를 사용하여 클론 코딩한 반응형 프로젝트입니다.

- 제작기간: 4일 (Refactor ~ing)
- 사용언어: HTML, SCSS, JavaScript
- 라이브러리: jQuery, GSAP, ScrollTrigger, SplitType
- 유형: 반응형
- URL: https://miraeae.github.io/lightroom-seoul

## 💡 Project Point
- GSAP를 활용한 다양한 인터렉션 구현
- SCSS를 활용한 코드의 재사용성, 가독성 향상
- SplitType을 활용한 텍스트 효과
- Space(전시장) 섹션의 스크롤 슬라이드 구현
